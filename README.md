
## 🔎 JavaEditChecker 🔎

### ❓ Info Box
JavaEditChecker is a very trivial program in python to check the SHA256 of all versions of minecraft

### 🚀 Getting Started

- Enter the suspect version
- Copy the SHA256 that will be generated and paste it to scan the version

### 🛠️Features

- Open Source
- Fast
- Safe
